******************************************************************************
Minimal (No GUI) CentOS Development Environment Preparetion steps 
	for Medyasoft PRODIGY
******************************************************************************

- 

Install Minimal CentOS 7
http://isoredirect.centos.org/centos/7/isos/x86_64/CentOS-7-x86_64-Minimal-1708.iso

Note : For virtual environment support ip connectevity with host and internet 
(VirtualBox : https://serverfault.com/questions/225155/virtualbox-how-to-set-up-networking-so-both-host-and-guest-can-access-internet)

- 

Login as root and upgrade the OS
$ yum upgrade

- 

Login as root and install fundemental development tools.
$ yum groupinstall 'Development Tools'
$ yum install readline-devel
$ yum install openssl-devel
$ yum install git
$ yum install vim
$ yum install wget 

- 

Configure git
$ git config --global user.name "Your Name"
$ git config --global user.email "you@example.com"

- 

Stop Firewall
$ systemctl disable firewalld
$ systemctl stop firewalld 

-

Install SFTP & SAMBA & SSH Connectivity
$ yum install vsftpd
https://www.liquidweb.com/kb/how-to-install-and-configure-vsftpd-on-centos-7/

$ yum install samba samba-client samba-common
https://www.howtoforge.com/samba-server-installation-and-configuration-on-centos-7

Connect via SSH & FTP from host (windows)

-

Create a postgres/postgres user/pwd
$ useradd postgres
$ passwd postgres
    -> postgres
$ usermod -g wheel postgres    (add user to sudors goup)

- 

download PostgresSQL sources && build.
git://git.postgresql.org/git/postgresql.git    (branch:)
$ mkdir postgresql
$ cd postgressql
$ git clone -b REL_10_STABLE git://git.postgresql.org/git/postgresql.git

- 

Build, Lunch & install PostgreSQL
https://www.postgresql.org/docs/current/static/install-procedure.html
$ mkdir build_dir
$ cd build_dir
$ ../configure [options go here]
$ make
$ sudo make install 

- 

Start PostgreSQL Server

add to .bash_profile
export LD_LIBRARY_PATH=/usr/local/pgsql/lib
export PATH=/usr/local/pgsql/bin:$PATH

$ sudo mkdir /usr/local/pgsql/data
$ sudo chown postgres /usr/local/pgsql/data
$ initdb -D /usr/local/pgsql/data --pwprompt 
$ pg_ctl -D /usr/local/pgsql/data -l /home/postgres/logfile start


edit /usr/local/pgsql/data/postgresql.conf
listen_addresses = '192.168.56.101'      (--> Server IP)
port = 5432

edit /usr/local/pgsql/data/pg_hba.conf
host    all             all             192.168.56.0/24         trust


Reload PostgreSQL server
$ pg_ctl -D /usr/local/pgsql/data -l /home/postgres/logfile reload
or
$ pg_ctl -D /usr/local/pgsql/data -l /home/postgres/logfile start
$ pg_ctl -D /usr/local/pgsql/data -l /home/postgres/logfile stop
